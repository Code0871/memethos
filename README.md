# Memethos

>⚠️ На стадии активной разработки

## Что это?
MemEthos — это эпизодическая база данных для памяти RAG (Retrieval-Augmented Generation) систем, работающая на векторах и графовых структурах

**Ключевые возможности:**
- Гибридный поиск (векторы + метаданные)
- Хронологическая релевантность
- Поддержка графовых связей между сущностями


## Как работает? 
Немного терминов для начала:
 - Moment (момент) — атомарная единица памяти (одно взаимодействие/событие)
 - Chain (цепочка) — упорядоченная последовательность moments, объединённых общим контекстом

В рамках системы Chain организовывается в виде точки в векторном пространстве, которая хранит набор Guid моментов, а также общие данные, которые нам нужны (например, тему цепочки, временные границы, агрегированные метаданные)

Moment же хранится отдельно — как собственный эмбеддинг с полезной нагрузкой (контент, метаданные и т.п.) и Guid-ссылкой на свою Chain. Таким образом:
 - поиск по Chain даёт быстрый "верхнеуровневый" ответ — к какому контексту вообще относится запрос;
 - поиск внутри найденной Chain по Guid-набору её моментов раскрывает конкретные детали.

Поверх этой структуры строится граф связей между моментами. Эти связи материализуются заранее, а **не** вычисляются на лету при каждом запросе, поэтому сборка графа для контекста происходит за **_O(1)_**
